# **App Name**: DataFolio Pro

## Core Features:

- Impactful Hero Section: Landing page with a hero section showcasing a photo, a clear title ('Transforming Data into Strategic Decisions'), and prominent call-to-action buttons ('View Projects' and 'Download CV').
- About Me Section: A concise 'About Me' section highlighting the blend of data science and full-stack development skills.
- Project Gallery CMS: An integrated CMS to manage projects. Each project includes a title, brief description, technologies used (tags like Python, SQL, React), a cover image, and a field to upload a PDF report.
- Project Detail Page: Projects are displayed in a grid. Clicking a project opens a detail page with a PDF preview and a 'Download Full Report' button.
- Visual Skills Section: Visually represent skills with progress bars or charts for Python (Pandas, Matplotlib), SQL, React, Node.js, and BI Tools.
- Contact Form with Database: A functional contact form connected to a database to receive messages from recruiters.
- Pre-populated Sample Projects: Three pre-populated project examples in the database: 'Netflix Content Analysis' (Python, Data Viz), 'Sentiment Analysis System' (Full Stack, NLP), and 'Corporate Financial Dashboard' (SQL, Excel).

## Style Guidelines:

- Primary color: Deep blue (#224369) for a trustworthy and corporate feel.
- Background color: Light grey (#F0F4F8), almost white, for a clean and minimalist aesthetic.
- Accent color: Steel grey (#6C7A89) for secondary elements and highlights, providing a subtle contrast.
- Body and headline font: 'Inter' sans-serif for a modern, clean, and readable look.
- Responsive design with a mobile-first approach for accessibility across all devices.
